About

The Library of Babel considers the library as an enduring metaphor for knowledge, information, and reading a world built entirely from symbols — vast enough to contain everything, and still nothing.

This project is a student collaboration with an LLM, typography, and exhibition design, situated at ArtCenter College of Design. The site functions as a speculative museum page for a fictional institution that takes Borges seriously as an interface brief.

All events, dates, and exhibitions are fictional, but the questions are not: what does it mean to navigate an archive that already contains every text you could write, and how can design make that abstraction feel tactile, local, and human-scaled?
